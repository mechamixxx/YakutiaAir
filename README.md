# Описание
Мобильное приложение для просмотра табло рейсов аэропорта города Якутск
Требовании:
  Android
Запуск:
  Скачать apk файл
  Устоновить apk файл
  Иметь подключение к интернету
Roadmap:
MVP
  Информация о рейсе - ✅
  Дата вылета/прилета - ✅
  Маршрут рейса - ✅

  Дизайн - ✅
  Информация о погоде - ✅
  Проверка работоспособности кнопок - ✅
